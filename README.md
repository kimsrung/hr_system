- Rails 5
- HTML5

- Bootstrap

##System dependencies

- postgres
- carrierwave
- devise

## Bundle and Run the Application
clone the project
```
$ git clone git@github.com:kimsrung/hr_system.git  
$ cd hr_system
```

bundle project
```
bundle install
```

initialization database
```
rake db:create db:migrate db:seed
```

Run Project
```
rails s
```

run the test
```
bundle exec rspec
```
